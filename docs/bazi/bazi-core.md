# 八字模块内核定位

本模块定位为“盲派读象式八字报告系统”，目标不是堆砌术语，而是把四柱、十神、藏干、宫位与用户问题转化为可阅读、可决策的命理咨询报告。

## 分析顺序

1. 四柱：先看年、月、日、时的整体分布。
2. 日主：确定分析参照点。
3. 十神：判断人事类型、性格倾向与资源关系。
4. 藏干：读取隐藏人事、内在资源、暗线机会。
5. 宫位：把十神落点转化为具体人生事项。
6. 干支关系：观察合冲刑害、生克流通。
7. 宾主：判断资源、人物与事件归属。
8. 制用：判断可用之神与需要约束的力量。
9. 墓库：观察资源收藏、阻滞与打开时机。
10. 大运流年：判断阶段趋势与关键年份。
11. 报告：输出普通用户能理解的结论、风险和行动建议。

## 版本范围

第一版先实现四柱、十神、藏干、宫位、报告模板，并用模拟规则生成关键年份与专业展示字段。

第二版再扩展合冲刑害、制用、墓库、大运流年，把占位字段逐步替换为真实算法。
