# 四柱宫位

## 年柱

年柱代表祖上、早年、外部环境、远方资源，也可看一个人与大环境、家族背景、早期资源的关系。

## 月柱

月柱代表父母、兄弟、事业平台、青年阶段、社会位置，是判断事业土壤、社会秩序和格局成败的重要位置。

## 日柱

日柱代表自己、配偶、婚姻宫、中年阶段、核心关系。日干为日主，日支常用于观察伴侣关系与亲密互动。

## 时柱

时柱代表子女、晚年、下属、未来发展、长期结果，也可看一个人后期成就与理想落点。

## 读法原则

宫位比单独十神更重要。十神落在哪个宫位，就要转化成对应的人生事项。例如财星落月柱，更多体现事业平台、资源调动与社会合作；财星落日支，则更容易进入亲密关系、生活安排和现实匹配。
